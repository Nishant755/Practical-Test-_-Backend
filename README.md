## env setup
    1)create .env file
    2)add following variables
        MONGODB_URI=mongodb://localhost:27017/habit-tracker
        PORT=3000
        JWT_SECRET=your-secret-key
## Package.json
    1)npm install   
    2)npm run dev {to run the server in development mode}
    3)npm run start {to start the server in production mode}
## Test
    1)npm run test  {to run the test}
    2)npm run test:watch {to run the test in watch mode}
    3)npm run test:coverage {to run the test with coverage}

## Docker
    1)npm run docker {to run the docker}
    2)npm run docker:stop {to stop the docker}
    3)npm run docker:down {to down the docker}
